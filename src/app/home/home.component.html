With NgRx

Current Count: {{ count$ | async }}


This app uses Angular Material, and has an example of a basic mobile-first responsive container class called "container" that is in contained in the styles.scss file, and used to wrap the router outlet that is in the app-component component.

There are 3 "Feature States" in this app; "app", "count", and "books". The "app" and "count" feature states follow their respective examples in links 3 and 4 below, from the RxJs web site. The third one, I did on my own. The pages for these are...

  • Home (this page) - Basic NgRx Project
  • User Profile (hamburger menu) - My Example
  • Books (hamburger menu) - Intermediate NgRx Project

Links 1 and 2 are for Youtube videos that explain what NgRx is and it's features. Link 1 is a short and high level description of NgRx. Link 2 is a deeper dive into how to write actions, reducers, and selectors. It gives a lot of information for the length of the video.

There are no examples of effects in this application... yet :)

On another note, there is a little css trick that i use to help debug responsive breakpoints, using the :after pseudo-element to display the resolution of the current breakpoint.
